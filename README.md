# 📄 Document Generator

Professional **offline document generator** for service projects fully custom for **PT Tidara Sabas** by Fajar Gumelar.
Creates polished Word documents (.docx) with Work Execution Plans (WEP), timelines, personnel, equipment lists, and more.

## ✨ Features

- 🎯 Work Execution Plan (WEP) generator with auto-formatting
- 👥 Personnel & Equipment management
- 📊 Timeline Gantt charts, Automatic S-Curve automation
- 📱 Fully offline
- 🏗 Native Windows desktop app (MSI installer)
